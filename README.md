# personal-dashboard

My personal dashboard extension
<h3>Steps on how to use it</h3>
<ol>
  <li>Go to chrome</li>
  <li>Click manage extension</li>
  <li>Open developer mode</li>
  <li>Click load unpacked on the upper left corner and select the folder of the project</li>
</ol>
